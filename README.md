📖 项目简介 (Introduction)

本项目是基于 HTML5 与 D3.js 开发的《理论力学》交互式思维导图。

本项目旨在通过可视化的知识网络，将静力学、运动学、动力学三大板块有机串联，帮助学生建立完整的知识体系。

本项目特别强调物理量纲的重要性，并融入了工程背景与趣味科学史，致力于让理论力学变得鲜活、立体。

✨ 核心功能 (Features)

📱 全平台适配：

电脑端：全景展示思维脉络，侧边栏显示详尽推导。

移动端：App 级交互体验，支持双指缩放、底部抽屉式弹窗与折叠卡片阅读。

📐 深度内容集成：

量纲分析：每个物理量均标注符号、国际单位制 (SI) 及量纲 (如 $MLT^{-2}$)。

工程意义：解析公式在机械设计、土木结构、航空航天等领域的实际应用。

课间冷知识：收录 $F=ma$ 的由来、转动惯量 $J$ 的选取原因等趣味背景。

🕸️ 知识关联：通过动态连线展示知识点之间的逻辑推导与合成关系。
